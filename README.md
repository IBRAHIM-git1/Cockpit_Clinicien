## How to edit and run this project locally

You can work on this project locally using your preferred IDE. The only requirement is Node.js (and npm).

Steps to run locally:

```powershell
# 1. Clone the repository
git clone <YOUR_GIT_URL>

# 2. Enter the project
cd <YOUR_PROJECT_NAME>

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Open `http://localhost:8080` in your browser to view the app.

## Editing files

## Technologies used

- Vite
- TypeScript
- React
- shadcn-ui (component primitives)
- Tailwind CSS

## Deployment

This project can be deployed to any static hosting provider that supports Vite-built apps (Netlify, Vercel, GitHub Pages, etc.). Build for production with:

```powershell
npm run build
```

Then serve the `dist` directory with a static host or use your provider's deployment integration.
